２．環境変数の設定
	環境変数 Springhead2 にこのフォルダのフルパスを指定する．
	[マイ コンピュータ]を右クリック→[プロパティ]→[詳細設定]ページ→[環境変数]
	→[新規]→ 変数名：	Springhead2		変数値:	C:\Home\svn\Project\Springhead2 など
