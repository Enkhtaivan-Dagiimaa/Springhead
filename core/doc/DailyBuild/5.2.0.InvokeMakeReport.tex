\subsection{スクリプトの起動}
\label{subsec:InvokeMakeReport}

MakeReportは、
\Path{<topdir>\BS core\BS test}を現在ディレクトリとして起動する。
関連するスクリプトファイル、データファイルについては
\KQuote{\Ref{subsec}{MakeReportScriptTree}}を参照のこと。

レポートファイルは、ディレクトリ\Path{.\BS report}に
\Path{YYYY-mmdd.report}という名前で作成される。
